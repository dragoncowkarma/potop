# 🎮 POTOP

**POTOP**은 1인칭 시점의 터렛 디펜스와 아케이드 슈터 요소가 결합된 고정 위치 360도 방어 게임입니다. 다가오는 적들을 물리치고 웨이브를 막아내며 생존하세요!

## 📌 게임 개요
- **장르**: 1인칭 터렛 디펜스 / 아케이드 슈터 (고정 위치 360도 방어)
- **개발 엔진**: Unity 6000.0.73f1
- **주요 기술**: C#, URP(Universal Render Pipeline), New Input System

## ✨ 주요 특징
- **긴장감 넘치는 웨이브 시스템**: 시간이 지나거나 적을 처치할수록 더 강력하고 다양한 적들이 몰려옵니다. (업데이트 예정)
- **다양한 적 유형**: 빠른 속도로 달려오는 적, 느리지만 체력이 높은 탱커형 적 등 전략적인 대처가 필요한 적들이 등장합니다. (업데이트 예정)
- **다양한 무기 시스템**: 기본 단발포 외에도 샷건, 3점사, 레이저 등 다양한 무기 시스템이 준비되어 있습니다. (업데이트 예정)

## 🛠 기술적 특징
안정적이고 확장 가능한 게임 환경을 위해 아래와 같은 아키텍처 패턴을 목표로 개발 중입니다.
- **이벤트 기반 통신 (Event-Driven Architecture)**: 스크립트 간 결합도를 낮추고 유연한 상호작용 구현
- **오브젝트 풀링 (Object Pooling)**: 메모리 최적화를 위해 적과 발사체를 효율적으로 관리
- **데이터 주도 설계 (Data-Driven Design)**: ScriptableObject를 활용하여 적, 무기, 웨이브 데이터를 구조적으로 분리 및 관리

## 🚀 현재 개발 상태 및 향후 계획
현재 **Phase 1 MVP(Minimum Viable Product)** 개발이 완료되어 있습니다.
- 핵심 게임 루프 구성 완료
- UI 및 씬 전환 기능 구현
- 기본 터렛 및 적 스폰 기능 구현

**향후 업데이트 예정 사항 (Phase 2 & 리팩토링)**:
- 시스템 아키텍처 고도화 (이벤트 시스템, 풀링, SO 전면 도입)
- 웨이브 기반 난이도 조절 시스템 구현
- 적 다양화 및 무기 확장
