# 🎮 POTOP (Project Over-The-Top Power)

**POTOP**은 360도로 몰려오는 적들을 격파하며 진화하는 **로그라이트 터렛 디펜스** 게임입니다. 고정된 자리에서 쏟아지는 화력을 만끽하고 최고의 랭킹을 차지하세요!

## 📌 게임 개요
- **장르**: 로그라이트 360도 웨이브 슈터
- **플랫폼**: Mobile, PC (Steam), VR (Meta Quest, SteamVR), Console (Nintendo Switch, PS5)
- **개발 엔진**: Unity 6000.0.73f1 (URP)
- **핵심 재미**: 압도적 화력 구축, 무작위 진화의 재미, 글로벌 랭킹 경쟁

## ✨ 주요 특징
- **웨이브 기반 로그라이트 성장**: 매 웨이브마다 주어지는 무작위 강화 선택지를 통해 자신만의 강력한 터렛 빌드를 완성합니다.
- **오버클럭(무한 모드)**: 15분의 웨이브와 보스전을 클리어한 후, 자신의 한계를 시험하는 무한 모드에 돌입하여 글로벌 리더보드에 이름을 올립니다.
- **다양한 적과 전략**: 스피드형, 탱커형, 자폭형 등 각기 다른 기믹을 가진 적들에 대응하는 최적의 무기 시너지를 찾아내세요.

## 🛠 기술 아키텍처 (Roadmap)
안정적이고 확장 가능한 개발을 위해 아래 시스템을 고도화 중입니다.
- **Event Broker**: 시스템 간 결합도를 낮춘 이벤트 중심 통신 아키텍처.
- **Object Pooling**: 수많은 투사체와 적을 효율적으로 처리하기 위한 최적화 기법.
- **Data-Driven Design**: ScriptableObject를 활용한 밸런스 및 웨이브 데이터 관리.
- **Global Leaderboard**: Firebase 연동을 통한 전 세계 유저 간의 실시간 점수 경쟁.
